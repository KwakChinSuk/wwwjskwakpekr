솔루션 특징

GA 데이터,
추가 태깅 없이 바로 활용하세요.

Alog는 Google Analytics(및 GA4) 데이터를 API로 자동 수집하고, 핵심 방문 지표를 직관적인 대시보드로 보여주는 솔루션입니다. 필요 시 BigQuery로 심화 분석까지 확장 가능합니다.

Alog – 솔루션 핵심 특징

추가 Tag 작업 없음

기존 GA(또는 GA4) 설치만 되어 있다면, 별도 이벤트 태깅 없이 바로 시작합니다. GA API Key 연동 후 곧바로 수집이 진행됩니다.

GA/GA4 지원 API 기반 빠른 도입

비교 지표 자동 계산

전일/전주/전월/전년 동기간을 자동 계산하여 변화 추세를 한눈에 파악합니다. 상승/하락, 연속 추세도 함께 강조합니다.

전일/전주/전월/전년비 추세 강조 경영 요약

맞춤형 대시보드

업무 목적에 맞춘 위젯 구성이 가능합니다. 방문 수, 페이지뷰, 활성 사용자, 재방문자, 전환 등 핵심 지표를 보기 좋게 정리합니다.

마케팅/전환 콘텐츠 경영 요약

AWS 기반 ASP

보안성과 가용성을 갖춘 클라우드 환경에서 안정적으로 운영됩니다. 추가 서버 구매나 전담 IT 인력 없이 구독 형태로 이용하세요.

안정성 보안 구독형

BigQuery 확장(선택)

유입/이탈, 경로, 사용자 변수 등 심화 분석이 필요할 때 BigQuery로 손쉽게 확장할 수 있습니다.

경로 분석 유입·이탈 사용자 변수
* BigQuery 사용 시 GCP 비용이 소액 발생할 수 있습니다.

간편한 도입 절차

단계내용
01. 상담목표 지표, 데이터 소스(GA/BigQuery) 확인
02. 연동GA API Key 설정 및 수집 테스트
03. 설계대시보드 위젯·비교 지표 정의
04. 운영AWS 기반 ASP로 서비스 시작

요금/플랜 요약

GA Only

GA 연동, 방문 지표(방문 수·PV·활성 사용자 등), 비교 지표 제공

Plus (GA + BigQuery)

유입·이탈·경로·사용자 변수 등 심화 분석, 간단한 비용으로 확장

Enterprise

자체 로그/마케팅 시스템 연동, 대시보드 커스터마이징 및 운영 지원

자주 묻는 질문

추가 태깅이 정말 필요 없나요?

네. 기존 GA(또는 GA4) 설치만으로 시작하며, API를 통해 데이터를 수집합니다.

BigQuery는 필수인가요?

아닙니다. GA Only로 시작 후, 심화 분석이 필요하면 Plus/Enterprise로 확장하세요.

인프라는 어떻게 운영되나요?

AWS 기반 ASP로 제공되며, 추가 서버 구매나 전담 IT 인력 없이 사용 가능합니다.
